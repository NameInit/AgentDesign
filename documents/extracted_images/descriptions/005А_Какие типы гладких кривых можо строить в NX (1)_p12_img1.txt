**КЛАССИФИКАЦИЯ:** 
Точный тип — интерфейс Siemens NX.

**СУТЬ И КОНТЕКСТ:**
Инженерный и геометрический смысл — это интерфейс программы для создания 3D-объектов. Здесь изображены инструменты и параметры, используемые для создания геометрических фигур и их модификации в программе.

**ДЕТАЛИЗАЦИЯ:**
Список инструментов и параметров:
- Параметр "Тип" — выбор типа геометрической фигуры (например, цилиндр, конус, куб и т. д.).
- Параметр "Распознание формы" — функция для автоматического распознавания формы в 3D-объекте.
- Параметр "Сопряжение" — инструмент для создания сопряжений (например, сферы, цилиндры и т. д.).
- Параметр "Параметризация" — инструмент для создания параметрических моделей.
- Параметр "Тело" — выбор типа тела (например, твердотельный, пористый и т. д.).
- Параметр "Сегмент" — инструмент для создания сегментов.
- Параметр "Параметр" — параметры для создания и модификации 3D-объекта.
