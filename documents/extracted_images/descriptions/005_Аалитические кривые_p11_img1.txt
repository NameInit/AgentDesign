*   САПР: Siemens NX.

### 2. СУТЬ И КОНТЕКСТ:

*   Объект: Конус.
*   Уравнение/закон: Уравнение конуса в 3D-пространстве.

### 3. ДЕТАЛИЗАЦИЯ:

*   Уравнение конуса: 
    \begin{equation*}
    \begin{cases}
    x^2 + y^2 = z^2 \quad \text{ (поверхность конуса)} \\
    z \geq 0 \quad \text{ (отрицательный конус)}
    \end{cases}
    \end{equation*}
*   Визуализация: 
    \begin{itemize}
    \item   Базовая окружность: $x^2 + y^2 = 1$.
    \item   Вектор нормали: $\vec{n} = \langle 0, 0, 1 \rangle$.
    \item   Вектор направления: $\vec{d} = \langle 0, 0, 1 \rangle$.
    \item   Центр: $C = (0, 0, 0)$.
    \item   Вершина: $V = (0, 0, 1)$.
    \item   Радиус: $r = 1$.
    \item   Высота: $h = 1$.
    \item   Эффективность: $\varepsilon = 1$.
    \item   Силуэтер: $\text{Sil} = \{ (x, y, z) \in \mathbb{R}^3 : x^2 + y^2 = z^2 \}$.
    \item   Классификация: Твердотельная геометрическая фигура.
    \item   САПР: Siemens NX.
    \item   Визуализация: 3D-чертеж проекций.
    \end{itemize}
