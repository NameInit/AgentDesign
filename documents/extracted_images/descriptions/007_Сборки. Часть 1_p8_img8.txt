Требуемый тип изображения: **Схема/алгоритм** в Siemens NX.

**Суть и контекст:**

Изображение представляет собой геометрическую фигуру с осью симметрии. Фигура состоит из двух полукругов, соединённых прямой линией. На фигуре указана ось симметрии.

**Детализация:**

1.  **Полукруги**:
    *   Каждый полукруг имеет одинаковую радиус и центр.
    *   Центр каждого полукруга совпадает с центром фигуры.
    *   Радиус каждого полукруга равен половине расстояния между центром фигуры и любой точкой на прямой линии.
2.  **Прямая линия**:
    *   Прямая линия соединяет центры двух полукругов.
    *   Длина линии равна двум радиусам.
3.  **Ось симметрии**:
    *   Ось симметрии проходит через центр фигуры и перпендикулярна прямой линии.
    *   Ось симметрии является осью вращения для фигуры.

**Математическая формула**:

Пусть:

*   $C_1$ — центр первого полукруга.
*   $C_2$ — центр второго полукруга.
*   $R_1$ — радиус первого полукруга.
*   $R_2$ — радиус второго полукруга.
*   $L$ — длина прямой линии.
*   $C$ — центр фигуры.

Формула для радиуса первого и второго полукруга:

$$R_1 = R_2 = \frac{L}{2}$$

Формула для расстояния между центрами:

$$C_1C_2 = 2R_1 = L$$

Формула для радиуса:

$$R = \frac{L}{2}$$

Формула для центра:

$$C = \frac{C_1 + C_2}{2}$$

Формула для оси:

$$O = \frac{C_1C_2}{2}$$

Формула для вектора:

$$\vec{v} = C_1C_2$$
