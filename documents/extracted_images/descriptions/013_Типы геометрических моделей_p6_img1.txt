### 1. КЛАССИФИКАЦИЯ

Точный тип: 2D геометрическая фигура.

### 2. СУТЬ И КОНТЕКСТ

Объект: аналитическая кривая.

Контекст: кривая сдвига.

Закон: векторное уравнение линии.

### 3. ДЕТАЛИЗАЦИЯ

Векторное уравнение линии:

$\mathbf{r} = \mathbf{r_0} + t \mathbf{v}$

*   $\mathbf{r}$ — вектор положения точки на линии.
*   $\mathbf{r_0}$ — вектор положения начала отсчета.
*   $\mathbf{v}$ — вектор направления линии.
*   $t$ — скалярный параметр, определяющий положение точки на линии.

Схема кривого сдвига:

*   Векторное уравнение кривого сдвига:

    $\mathbf{r} = \mathbf{r_0} + t \mathbf{v} + \mathbf{w}$
*   Векторное уравнение линии:

    $\mathbf{r} = \mathbf{r_0} + t \mathbf{v}$
*   Векторное уравнение кривого сдвига относительно линии:

    $\mathbf{w} = \mathbf{r} - \mathbf{r_0} - t \mathbf{v}$

Векторное уравнение кривого сдвига:

$\mathbf{r} = \mathbf{r_0} + t \mathbf{v} + \mathbf{w}$

*   $\mathbf{r}$ — вектор положения точки на кривом сдвиге.
*   $\mathbf{r_0}$ — вектор положения начала отсчета.
*   $\mathbf{v}$ — вектор направления кривого сдвига.
*   $\mathbf{w}$ — вектор кривого сдвига.
*   $t$ — скалярный параметр, определяющий положение точки на кривом сдвиге.

Координаты кривого сдвига:

*   $x = x_0 + v_{x} t + w_{x}$
*   $y = y_0 + v_{y} t + w_{y}$
*   $z = z_0 + v_{z} t + w_{z}$

Векторы:

*   Векторы кривого сдвига: $\mathbf{w} = (w_{x}, w_{y}, w_{z})$
*   Векторы линии: $\mathbf{v} = (v_{x}, v_{y}, v_{z})$
*   Векторы начала: $\mathbf{r_0} = (x_0, y_0, z_0)$
