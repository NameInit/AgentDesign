Вершины: (0,2) и (0,-2)

Фокусы: (1,0) и (-1,0)

Оси симметрии: x и y

Парабола: 

\begin{equation*}
y^2 = 4x
\end{equation*}

Код на языке Python:

```python
import matplotlib.pyplot as plt
import numpy as np

# создаем массивы для x и y
x = np.linspace(-2, 2, 100)
y = np.sqrt(4*x)

# создаем график
plt.plot(x, y)
plt.plot(x, -y)
plt.title('Парабола')
plt.xlabel('x')
plt.ylabel('y')
plt.grid(True)
plt.show()
```
