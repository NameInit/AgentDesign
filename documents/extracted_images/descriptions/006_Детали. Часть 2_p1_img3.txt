*   Схема: нет

**СУТЬ И КОНТЕКСТ:**

*   Объект: парабола
*   Уравнение: `y = 2x^2 - 3x + 1`
*   Аналитическая кривая: да
*   Твердотельная операция: нет
*   Сопряжение: нет
*   Метод начертательной геометрии: нет
*   Полигональная сетка: нет

**ДЕТАЛИЗАЦИЯ:**

*   Уравнение параболы: `y = 2x^2 - 3x + 1`
*   Основной параметр: `a = 2` (коэффициент квадратичной функции)
*   Сдвиг по x: `b = -3` (коэффициент линейной функции)
*   Сдвиг по y: `c = 1` (коэффициент постоянного члена)
*   Начало: `x = -b / 2a = 3/4`
*   Верхняя точка: `y = 2(x - 3/4)^2 + 1`
*   Дно: `y = -2(x - 3/4)^2 + 1`

Вывод: Визуализация представляет собой параболу с уравнением `y = 2x^2 - 3x + 1`, с основным параметром `a = 2`, сдвигом по x `b = -3` и сдвигом по y `c = 1`. Начало параболы находится в точке `x = -b / 2a = 3/4`, а ее верхняя и нижняя точки определяются по уравнению. Это кривая, которая вписывается в рамки инженерной геометрии и САПР, и может быть представлена в САПР-инструменте, таком как Siemens NX.
