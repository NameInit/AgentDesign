**КЛАССИФИКАЦИЯ**

*   Формула: 3D геометрическая фигура (тело)
*   Твердотельная операция: вырезание из твердого тела (выполнение операции в пространстве)

**СУТЬ И КОНТЕКСТ**

*   Описывает процесс вырезания из твердого тела с помощью операции "Вырезать" в моделировании CAD.
*   Операция "Вырезать" предполагает удаление определенной геометрической фигуры из исходного твердого тела, оставляя новую геометрическую фигуру.
*   Применяется для создания сложных геометрических фигур путем объединения операций вырезания и объединения.

**ДЕТАЛИЗАЦИЯ**

*   Операция "Вырезать" имеет два основных параметра:
    *   **Средняя геометрическая фигура**: это фигура, которую необходимо вырезать из исходного тела.
    *   **Исходная геометрическая фигура**: это фигура, из которой вырезается средняя фигура.
*   Операция "Вырезать" выполняется по алгоритму:
    *   Выполнение операции "Выполнить" для средней геометрической фигуры.
    *   Выполнение операции "Объединить" для средней и исходной геометрических фигур.
*   Результатом операции "Вырезать" становится новая геометрическая фигура, которая представляет собой вырезанный из исходного тела объем.

**ОБЪЕКТЫ И ТЕРМИНЫ**

*   **Средняя геометрическая фигура**: это фигура, которую необходимо вырезать из исходного тела.
*   **Исходная геометрическая фигура**: это фигура, из которой вырезается средняя фигура.
*   **Операция "Выполнить"**: выполняет геометрическую операцию для средней фигуры.
*   **Операция "Объединить"**: объединяет среднюю и исходные фигуры.
*   **Новая геометрическая фигура**: представляющая собой вырезанный из исходного тела объем.
